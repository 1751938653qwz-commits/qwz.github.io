Work-Life Boundary · 工作与生活边界
====================================

一个基于摄像头人体识别的互动网页艺术项目。
职场文字像雨一样从屏幕落下，你的身体可以阻挡它们。

功能说明
--------
1. 权限确认页面（8项权限，全部允许后进入主页面）
2. 摄像头识别人体（TensorFlow.js + BodyPix）
3. 职场文字从顶部匀速下落（中英文混合）
4. 身体阻挡文字（碰到人体轮廓即停止）
5. 保持不动超过8秒 → 弹出 "Continue working" 提示框
6. 办公消息提示音（Web Audio API 合成）

使用方法
--------
方式一：直接打开文件
  双击 index.html 即可在浏览器中打开

方式二：部署到网络（任何人可访问）
  将整个文件夹上传到任意静态托管服务：
  - GitHub Pages（免费）
  - Vercel（免费）
  - Netlify（免费）
  - 自己的 Web 服务器

技术要求
--------
- 现代浏览器（Chrome / Edge）
- 需要摄像头
- 需要网络（加载 TensorFlow.js 和 BodyPix 模型）

文件说明

index.html  - 主页面（所有代码都在这个文件中）
